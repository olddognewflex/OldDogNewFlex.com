## Product Line Overview

**Responsible AI Adult** is a suite of production-safety prompt frameworks for AI-assisted software engineering.

The unifying principles:

- Tool-agnostic (ChatGPT, Claude, Copilot, Cursor, local LLMs)
- Markdown-first, portable, inspectable
- Bias toward safety, incrementalism, and survivability
- Encodes senior engineering judgment, not AI hype
